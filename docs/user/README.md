# User Documentation Hub

This folder contains end-user documentation for using the optimization interface and understanding how problem modules are implemented at a practical level.

These guides are written for participants and operational users. They focus on workflow, configuration, and interpretation of results.

## Start Here

1. `INTERFACE_GUIDE.md`  
   How to use chat, definition/config tabs, optimization runs, and result interpretation.

2. `PROBLEM_MODULES_GUIDE.md`  
   How problem modules are structured, how weights/constraints map to solver behavior, and what users can configure.

3. `ASKING_THE_AGENT.md`  
   Suggested prompts users can ask the assistant (including "what did you program?") and what kind of answers to expect.
